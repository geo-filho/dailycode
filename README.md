# 🚀 DailyCode

O **DailyCode** é um desafio de programação que eu criei com o objetivo de incentivar o aprendizado contínuo — tanto para mim quanto para meus amigos.

A ideia é simples: **praticar um pouco todos os dias**, de forma leve, consistente e colaborativa.

---

## 🎯 Objetivo

Criar o hábito de programar diariamente, melhorar o raciocínio lógico e evoluir juntos como desenvolvedores.

Mais importante que velocidade ou perfeição, aqui o foco é:
- consistência 📅  
- aprendizado 📚  
- colaboração 🤝  

---

## 📅 Como funciona

- Todos os dias são propostos **exercícios de programação**  
- O tempo ideal é de **até 30 minutos por dia**  
- Cada participante resolve os exercícios no seu ritmo  
- As soluções devem ser **enviadas para o repositório até as 20h**  

O desafio é por tempo indeterminado — vamos seguir até:
- finalizar a lista de exercícios  
ou  
- decidirmos encerrar 😄  

Além disso, temos um **desafio bônus semanal**, com problemas um pouco mais abertos ou criativos.

---

## 📜 Regras

### 🔎 Uso de pesquisa
Você pode (e deve) pesquisar, mas siga essa ordem:

1. Tente resolver sozinho  
2. Busque documentação (ex: W3Schools)  
3. Use IA se necessário  

---

### 🤖 Uso de IA
Permitido, mas com responsabilidade:

> **Você deve entender o código linha por linha.**  
> IA é **copilot**, não **piloto**.

---

### 💻 Linguagem
Apesar dos exercícios serem baseados em Python,  
**você pode usar qualquer linguagem**.

---

### 🤝 Colaboração
- É permitido (e incentivado) pedir ajuda  
- Compartilhar conhecimento faz parte do desafio  

---

### ⏰ Entrega
- Subir os exercícios no GitHub  
- Prazo: **até 20h do dia**

---

## 📁 Estrutura do repositório

Acreditamos que pequenas práticas diárias geram grandes resultados no longo prazo.

Este desafio não é sobre competição, e sim sobre:
- evoluir juntos  
- criar disciplina
